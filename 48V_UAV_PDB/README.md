# 48V UAV Power Distribution Board

## Overview
Designed a high-current (120–150A) power distribution board for 40–60V UAV systems.

## Features
- Wide input range: 40–60V
- Multiple regulated outputs (12V, 5V)
- Reverse polarity protection
- TVS-based transient suppression

## Design Highlights
- High-current PCB layout considerations
- Thermal management strategies
- Noise and transient handling for ESC loads
